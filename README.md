# Web Uno Arena - File Structure

## Project Layout

### 📁 public/
Static assets served at the root domain (`/`).
- `sw.js`: Service Worker for PWA and Monetag verification.
- `manifest.json`: Web app manifest for PWA installation.
- `googlee3a7cef794a737a8.html`: Google Search Console verification file.
- `favicon.ico`: (Auto-generated by browser/Next.js)

### 📁 src/
#### 📁 ai/
- `genkit.ts`: AI initialization.
- `flows/ai-strategic-hint-tool.ts`: Logic for AI-powered gameplay hints.

#### 📁 app/
- `layout.tsx`: Root layout containing Monetag Ad scripts and global fonts.
- `page.tsx`: The main landing page/lobby where players enter their names and join rooms.
- `globals.css`: Tailwind styles and custom mesh gradients.
- 📁 `game/[roomId]/`:
    - `page.tsx`: The main Game Arena where combat happens.

#### 📁 components/
- 📁 `ui/`: Shared ShadCN components (Button, Input, Dialog, etc.).
- 📁 `uno/`: Custom Uno-specific components:
    - `UnoCardUI.tsx`: 3D-styled Uno cards.
    - `ChatSidebar.tsx`: Real-time chat with notification logic.
    - `WildColorPicker.tsx`: Color selection overlay for Wild cards.
    - `UnoButton.tsx`: The "UNO!" shout button.

#### 📁 lib/
- `uno-engine.ts`: Core game logic (deck creation, card validation, penalties).
- `placeholder-images.ts`: Avatar management.

#### 📁 firebase/
- `index.ts`: Firebase initialization.
- `provider.tsx`: Context provider for Firestore/Auth.
- 📁 `firestore/`: Custom hooks for real-time document and collection syncing.

#### 📁 hooks/
- `use-toast.ts`: UI notification system.
- `use-mobile.tsx`: Responsive design detection.

## Key Features
- **Multiplayer**: Real-time sync via Firestore.
- **PWA**: Installable on mobile devices.
- **Monetag Ads**: Integrated vignette ads.
- **AI Hints**: Strategic suggestions powered by Google Gemini.
